# Contributing

欢迎提交问题和改进建议。

## 提交 Issue

请尽量提供：

- 软件版本。
- Windows 版本。
- 是否已安装 R。
- 输入文件格式。
- 错误截图或启动器日志。

## 本地开发

1. 安装 R。
2. 运行 `install_dependencies.R`。
3. 运行 `test_app.R`。
4. 修改 `app.R` 或 `DifferentialAppLauncher.cs`。

## 打包

重新编译启动器后，将需要发布的文件放入 `dist/BioInsight-OneClick-Bioinformatics-v1.2.0.zip`。
