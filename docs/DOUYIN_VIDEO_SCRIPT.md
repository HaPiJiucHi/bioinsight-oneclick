# 抖音讲解视频脚本

目标：吸引生信入门用户、课题组学生、不会写 R 但需要做表达谱差异分析的人下载 GitHub Release。

## 60 秒版本

### 0-3 秒：钩子

画面：打开软件首页。

旁白：

> 不会写 R，也能一键做差异分析、火山图、热图、PCA、GSEA、WGCNA 和 PPI。

屏幕字幕：

```text
表达矩阵导入后，一键出差异分析结果
```

### 4-12 秒：导入数据

画面：左侧导入表达矩阵、分组表、注释表。

旁白：

> 左边只需要导入表达矩阵和分组表。表达矩阵是行代表基因，列代表样本；分组表只要 sample 和 group 两列。

屏幕字幕：

```text
表达矩阵 + 分组表 + 可选注释表
```

### 13-25 秒：基础结果

画面：点击“一键开始差异分析”，切到火山图、热图、PCA。

旁白：

> 点一下开始分析，软件会自动跑 limma，直接生成差异基因表、火山图、热图和 PCA。PCA 还能打开分组椭圆和中心点。

屏幕字幕：

```text
DEG / Volcano / Heatmap / PCA
```

### 26-45 秒：机制解释

画面：切到 GSEA、WGCNA、PPI 页签。

旁白：

> 如果你想写结果解释，可以继续看 GSEA、WGCNA 和 PPI。GSEA 看整套基因排序偏向哪些通路，WGCNA 看共表达模块偏向哪一组，PPI 用来找后续实验可以优先验证的 hub genes。

屏幕字幕：

```text
GSEA 解释通路
WGCNA 解释模块
PPI 筛选 hub genes
```

### 46-55 秒：示例结论

画面：展示 GSEA 图和 WGCNA 图。

旁白：

> 这个示例数据里，GSEA 提示细胞周期和染色体分离相关过程有明显整体偏向，WGCNA 也能看到 Disease 和 Normal 各自对应的模块。

屏幕字幕：

```text
不只是出图，还能形成结果解释
```

### 56-60 秒：引导下载

画面：GitHub README 和 Release 下载位置。

旁白：

> 软件已经放在 GitHub，Release 里直接下载 zip。需要的话可以 Star 一下，后面继续更新更多生信流程。

屏幕字幕：

```text
GitHub 搜：differential-analysis-software
Release 下载 zip
```

## 拍摄建议

- 前 3 秒必须展示最终图，不要先讲安装。
- 鼠标操作尽量慢一点，重点停留在页签切换和结果图。
- 每个图只讲一句用途，不要把统计原理讲太长。
- 结尾一定展示 GitHub Release 下载位置。
