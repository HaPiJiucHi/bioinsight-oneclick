# 常见问题

## 没有 R 能运行吗？

可以。双击 `差异分析软件.exe` 后，点击“检查依赖”，软件会把 R 安装到软件同目录的 `R` 文件夹。

## 为什么要联网？

第一次缺少 R 或 R 包时需要联网下载。依赖安装完成后，日常分析通常不需要联网。

## 表达矩阵应该怎么放？

第一列放基因 ID 或探针 ID，后面每一列是一个样本。样本列必须是数值。

## 分组表匹配不上怎么办？

检查分组表里的 `sample` 是否和表达矩阵列名完全一致，包括大小写、空格和特殊字符。

## 火山图没有标注基因名？

请确认：

- “火山图标注显著基因”已勾选。
- “标注前 N 个显著基因”大于 0。
- 当前阈值下确实有显著差异基因。

## 热图没有显示很多基因？

热图默认只展示 Top 差异基因，避免图片过密。可以调整“热图基因数”。
