# 常见问题

## 没有 R 能运行吗？

可以。双击 `差异分析软件.exe` 后，点击“检查依赖”，软件会把 R 安装到软件同目录的 `R` 文件夹。

## 为什么要联网？

第一次缺少 R 或 R 包时需要联网下载。依赖安装完成后，日常差异分析通常不需要联网。GSEA 使用本地安装的注释包，PPI 默认使用同目录的 `string_interactions.tsv`。

## 表达矩阵应该怎么放？

第一列放基因 ID 或探针 ID，后面每一列是一个样本。样本列必须是数值。

## 分组表匹配不上怎么办？

检查分组表里的 `sample` 是否和表达矩阵列名完全一致，包括大小写、空格和特殊字符。

## 火山图没有标注基因名？

请确认：

- “火山图标注显著基因”已勾选。
- “标注前 N 个显著基因”大于 0。
- 当前阈值下确实有显著差异基因。

## 热图没有显示很多基因？

热图默认只展示 Top 差异基因，避免图片过密。可以调整“热图基因数”。

## GSEA 出现部分基因无法映射的警告怎么办？

这是常见情况，尤其是芯片探针或历史基因名。只要仍有足够基因成功映射，GSEA 结果可以继续参考。若映射失败比例很高，建议检查注释表是否为人类 gene symbol，或更换对应物种注释。

## PPI 没有边怎么办？

可以降低 `combined_score` 阈值，或增加“PPI 最多差异基因数”。也要确认上传的 STRING interaction 文件与当前基因 symbol 属于同一物种和同一命名体系。

## WGCNA 结果应该怎么解释？

看模块与分组的相关性。相关性大于 0 表示模块整体更偏向处理组，相关性小于 0 表示更偏向对照组。WGCNA 解释的是模块级共表达结构，不等同于单个基因的差异分析结论。
