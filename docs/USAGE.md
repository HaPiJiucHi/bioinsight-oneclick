# 使用说明

## 1. 导入数据

表达矩阵应为“行=基因/探针，列=样本”。第一列是 ID，其余列是数值表达量。

支持格式：

- `.csv`
- `.tsv`
- `.txt`
- `.xlsx`
- `.xls`

## 2. 设置分组

有三种方式：

- 分组文件：推荐方式，至少包含 `sample` 和 `group` 两列。
- 样本名关键词：通过样本名里的关键词自动识别 Control/Treatment。
- 手动粘贴：直接粘贴 `sample,group` 表。

样本名必须和表达矩阵列名一致。

## 3. 设置差异分析参数

主要参数：

- `log2 转换`：自动判断、不转换、强制 `log2(x + 1)`。
- `样本间分位数标准化`：适合部分芯片表达矩阵。
- `log2FC 阈值`：默认 1。
- `P 值阈值`：默认 0.05。
- `显著性列`：可选 `P.Value` 或 `adj.P.Val`。

点击“一键开始差异分析”后，软件会生成差异表、火山图、热图和 PCA。

## 4. 图形设置

火山图：

- 可调整上调、下调、稳定基因颜色。
- 可勾选“火山图标注显著基因”。
- 可设置标注前 N 个显著基因。

热图：

- 可选择是否进行行聚类。
- 可选择是否进行列聚类。
- 可调整低值、中间值、高值颜色。

## 5. 高级分析

### WGCNA

WGCNA 用于寻找与分组相关的共表达模块。软件会把处理组编码为 1、对照组编码为 0，输出每个模块与分组的相关性、P 值、偏向组别和模块 hub genes。

适合回答：“哪些成组共表达基因模块更偏向疾病组/处理组？”

注意：WGCNA 对样本量敏感，样本太少时只建议作为探索性分析。

### PPI

PPI 会从显著差异基因里筛选候选基因，并基于 STRING 互作表构建蛋白互作网络，输出 degree、betweenness、closeness 等中心性指标。

适合回答：“哪些差异基因在已知/预测互作网络里更核心，值得优先验证？”

默认读取软件同目录下的 `string_interactions.tsv`。也可以上传自己的 STRING interaction 文件，建议至少包含 `node1`、`node2`、`combined_score`。

### GSEA

GSEA 会使用全部可排序基因，而不是只使用过阈值的 DEG。软件目前支持 GO `BP`、`MF`、`CC`。

适合回答：“整套表达变化整体偏向哪些生物过程或功能方向？”

GSEA 的好处是能识别一组基因整体轻中度同向变化，不容易被单个阈值截断信息。差异分析之后，建议优先看 GSEA 来解释机制。
