# 使用说明

## 1. 导入数据

表达矩阵应为“行=基因/探针，列=样本”。第一列是 ID，其余列是数值表达量。

支持格式：

- `.csv`
- `.tsv`
- `.txt`
- `.xlsx`
- `.xls`

## 2. 设置分组

有三种方式：

- 分组文件：推荐方式，至少包含 `sample` 和 `group` 两列。
- 样本名关键词：通过样本名里的关键词自动识别 Control/Treatment。
- 手动粘贴：直接粘贴 `sample,group` 表。

样本名必须和表达矩阵列名一致。

## 3. 设置参数

主要参数：

- `log2 转换`：自动判断、不转换、强制 `log2(x + 1)`。
- `样本间分位数标准化`：适合部分芯片表达矩阵。
- `log2FC 阈值`：默认 1。
- `P 值阈值`：默认 0.05。
- `显著性列`：可选 `P.Value` 或 `adj.P.Val`。

## 4. 图形设置

火山图：

- 可调整上调、下调、稳定基因颜色。
- 可勾选“火山图标注显著基因”。
- 可设置标注前 N 个显著基因。

热图：

- 可选择是否进行行聚类。
- 可选择是否进行列聚类。
- 可调整低值、中间值、高值颜色。

## 5. 下载结果

分析完成后可下载：

- 完整差异表
- 显著差异基因表
- 火山图 PNG
- 热图 PNG
- PCA PNG
