# 使用说明

## 1. 准备输入文件

最少需要表达矩阵和分组信息。新手可以在左侧下载模板；如果数据来自 GEO，可以上传 `series_matrix.txt.gz`，让软件先拆出表达矩阵和样本信息。

表达矩阵格式：

```csv
feature_id,Normal_1,Normal_2,Disease_1,Disease_2
GeneA,8.1,8.3,10.2,10.0
GeneB,5.0,5.2,4.8,4.7
```

分组表格式：

```csv
sample,group
Normal_1,Normal
Normal_2,Normal
Disease_1,Disease
Disease_2,Disease
```

## 2. 导入数据

左侧支持：

- 上传表达矩阵。
- 载入当前示例数据。
- 下载表达矩阵模板和分组表模板。
- 上传 GEO `series_matrix.txt.gz` 并拆出表达矩阵和样本信息。
- 上传注释表，注释表可选。

## 3. 设置分组

有四种方式：

- 分组文件：推荐方式，至少包含 `sample` 和 `group` 两列。
- GEO 样本信息列：拆出 GEO 样本信息后，选择一列生成分组表。
- 样本名关键词：通过样本名里的关键词自动识别 Control/Treatment。
- 手动粘贴：直接粘贴 `sample,group` 表。

样本名必须和表达矩阵列名一致。

## 4. 设置 DEG 参数

差异分析参数放在“分析结果”页签顶部。主要参数：

- `log2 转换`：自动判断、不转换、强制 `log2(x + 1)`。
- `样本间分位数标准化`：适合部分芯片表达矩阵。
- `log2FC 阈值`：默认 1。
- `P 值阈值`：默认 0.05。
- `显著性列`：可选 `P.Value` 或 `adj.P.Val`。

点击“一键开始分析”后，软件会生成差异表、火山图、热图和 PCA。

## 5. 图形设置

火山图：

- 可调整上调、下调、稳定基因颜色。
- 可标注前 N 个显著基因。

热图：

- 可选择是否进行行聚类。
- 可选择是否进行列聚类。
- 可调整低值、中间值、高值颜色。

PCA：

- 可选择是否显示分组椭圆。
- 可选择是否显示分组中心点。

## 6. 高级分析

GSEA 用全部可排序基因解释通路整体偏向，适合放在 DEG 之后解释机制。

WGCNA 用于寻找与分组相关的共表达模块，样本量太少时建议作为探索性结果。

PPI 用于从显著差异基因中筛选互作网络里的候选 hub genes。

## 7. 下载结果

分析完成后可下载：

- 完整差异表。
- 显著差异基因表。
- 火山图 PNG。
- 热图 PNG。
- PCA PNG。
- GSEA GO 结果表。
- WGCNA 模块表和 hub genes。
- PPI hub genes 和 edges。
